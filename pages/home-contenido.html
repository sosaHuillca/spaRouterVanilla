Herramientas Para Web

Html Es la estructura basica de la web, define la manera de ordenar el contenido sea texto, imagen, video, juegos, etc. los archivos tienen la extension '.html'

Css podemos definir las fuentes, colores, el disenio de la pagina. los archivos tienen la extension '.css'

Javascript Es un lenguaje de programacion, para poder darle mejor interaccion del lado del cliente.
